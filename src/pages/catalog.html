Винтовка Мосина

Популярные ложи для винтовки Мосина и ее модификаций (КО-44, 91/30). Данный раздел каталога содержит все виды ложи для Мосина. Три основных типа лож:

  • Тип СВД — главное преимущество данного типа, отличный внешний вид, пистолетная рукоять, крепление для ремня и малый вес за счет «скелетного» исполнения. Данный тип универсальный, подойдет как для левши, так и для правши.
  • Ортопедический тип — пистолетная рукоять, подщечник, резиновый затыльник, адаптация под левшу или правшу. Самая популярная модель ложа за последние 2 года.
  • Тип «Монте-Карло» — традиционный вид ложа, проверенный временем дизайн с полу пистолетной рукоятью подойдет для тех, кому нужен комфорт ортопедического ложа с сохранностью классического внешнего вида.
